# Kendrick Lamar Super Bowl Template

## Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/your-username/super_bowl.git
   cd super_bowl
   ```

2. **Install dependencies:**
   ```sh
   npm install
   ```

3. **Start the development server:**
   ```sh
   npm start
   ```

4. **Open your browser and visit:**
   ```
   http://localhost:3000
   ```
